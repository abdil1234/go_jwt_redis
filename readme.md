Go Simple Features : 
-----------------
 - crud rest api
 - redis caching
 - jwt auth
 - middleware with gorilla
 - working with gorm

INSTALLATION : 
-----
 - brew install dep
 - cd $GOPATH/src/github.com/me/go_jwt_redis
 - dep ensure

USAGE : 
-----

### GET TOKEN

```
    curl -X POST --user username:password http://localhost:8081/login
```
